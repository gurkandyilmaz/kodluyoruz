## Installation
Projeyi indirmek icin;
```bash
git clone https://github.com/gurkandyilmaz/kodluyoruz.git
```

## Usage
Projeyi indirdikten sonra herhangi bir editorde aciniz. Example: VSCode
```linux
cd kodluyoruz/
code .
```
